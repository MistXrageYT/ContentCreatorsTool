# 🛠️ ContentCreatorsTool

Welcome to the official **ContentCreatorsTool** Issue Tracker!

**ContentCreatorsTool** is a creator-focused Minecraft modpack designed for **content creators, builders, map makers, and technical players**. It combines powerful building tools, recording utilities, performance optimizations, quality-of-life improvements, and carefully selected resource packs into one easy-to-use package.

This repository is dedicated to:

* 🐞 Reporting bugs and crashes
* 💡 Suggesting new features
* 📝 Providing feedback
* 📈 Helping improve the modpack

# 🐞 Reporting Bugs

Before creating an issue, please make sure you:

* ✅ Are using the latest version of ContentCreatorsTool.
* ✅ Search existing issues before creating a new one.
* ✅ Test without adding extra mods or resource packs (if possible).
* ✅ Include your latest log or crash report when reporting crashes.

When reporting a bug, please include:

* ContentCreatorsTool Version
* Minecraft Version
* Java Version
* Operating System
* Launcher (Modrinth, Prism, CurseForge, Official Launcher, etc.)
* Description of the issue
* Steps to reproduce
* Crash Report or `latest.log`
* Screenshots (if applicable)
