# Changelog

All notable changes to **ContentCreatorsTool** will be documented in this file.

The format is based on **Keep a Changelog**, and versions follow the Minecraft version followed by the modpack version.

---

# [26.2-1.0.0] - Stable

### 🎉 Stable Release

ContentCreatorsTool has officially graduated from beta and is now considered stable for Minecraft 26.2.

### Added

* Bedrock Hotbar
* Locator Heads

### Updated

* Updated all bundled mods
* Updated resource packs
* Updated shaders
* Improved compatibility and overall stability

### Changed

* General optimization improvements
* Better compatibility with the Minecraft 26.2 mod ecosystem

### Removed

* Krypton *(temporarily)*
* FastQuit *(temporarily)*
* Fabrishot *(temporarily)*
* Cherished Worlds *(temporarily)*
